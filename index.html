关于我们

PythonStudy 是一个面向零基础学习者的交互式 Python 教学平台，共 26 个课时，涵盖变量、条件判断、循环、函数、类、装饰器等核心主题。

功能特点

  • 浏览器内直接编写和运行 Python 代码，无需安装任何软件
  • 基于 Pyodide，在浏览器中运行完整的 Python 解释器
  • 每课时包含概念讲解和配套习题，边学边练
  • 学习进度自动保存到本地浏览器

技术栈

  • Python 运行时：Pyodide (CPython compiled to WebAssembly)
  • 代码编辑器：CodeMirror 5
  • 前端：原生 HTML / CSS / JavaScript，无框架依赖
